**题目描述：**

给你一个字符串 s，找到 s 中最长的回文子串。

如果字符串的反序与原始字符串相同，则该字符串称为回文字符串。

**示例：**

示例 1：

```
pythonCopy code
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。

```

示例 2：

```
pythonCopy code
输入：s = "cbbd"
输出："bb"

```

**提示：**

1. 1 <= s.length <= 1000
2. s 仅由数字和英文字母组成











**解题思路：**

使用动态规划求解。

```python
def longest_palindromic_substring(s: str) -> str:
    n = len(s)
    if n < 2:
        return s

    dp = [[False] * n for _ in range(n)]
    for i in range(n):
        dp[i][i] = True

    max_len = 1
    start = 0
    for j in range(1, n):
        for i in range(j - 1, -1, -1):
            if s[i] == s[j]:
                if j - i < 3:
                    dp[i][j] = True
                else:
                    dp[i][j] = dp[i + 1][j - 1]
            else:
                dp[i][j] = False

            if dp[i][j]:
                cur_len = j - i + 1
                if cur_len > max_len:
                    max_len = cur_len
                    start = i

    return s[start: start + max_len]

```





**使用示例：**

```python
s1 = "babad"
s2 = "cbbd"

print(longest_palindromic_substring(s1))  # 输出: "bab" 或 "aba"
print(longest_palindromic_substring(s2))  # 输出: "bb"
```
